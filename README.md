# 3D Stair-Step Birthday Scroll

A full-screen, dark-themed birthday web experience where each section moves like a 3D staircase while scrolling.

## Features

- Full-screen sticky sections with stair-step parallax motion
- Smooth 3D rotation and depth effect tied to scroll position
- Bold uppercase typography using **Montserrat** from Google Fonts
- Distinct premium gradient backgrounds per message section
- Mobile-responsive layout and reduced-motion support

## Sections

- HAPPY
- BIRTHDAY
- TO YOU
- ALEX

## Run locally

Open `index.html` in any modern browser.

To customize the name, edit the final section text in `index.html`.
